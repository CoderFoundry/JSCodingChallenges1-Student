FizzBuzz Ternary

Write a program that prints the numbers from 1 to 100 and for multiples of ‘3’ print “Fizz” instead of the number and for the multiples of ‘5’ print “Buzz”. Print “FizzBuzz” for numbers which are multiples of both 3 and 5.

Implement your function using a ternary operator. Your method should return an array of values. Display the array of values on the page.

Extra credit: Allow the user to vary the range of numbers displayed and the fizz and buzz values.

Example Output: 1 2 Fizz 4 Buzz Fizz 7 8 Fizz Buzz 11 Fizz 13 14 FizzBuzz …
